Detailed Description

Represent the date of marks were obtained.

Date class contains 3 different variables. m_day, m_month and m_year are all int type variable. This class also contains appropriate get/set functions and operator. Inside the constructor and set functions also contains appropriate validation check to ensure the user enter valid date.
