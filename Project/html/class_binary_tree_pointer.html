Detailed Description

template<class T>
class BinaryTreePointer< T >

Represent the binary search tree.

This bst template class contains structure Node as member variables. The data type depends on the type passed in by the user. This class also contain necessary bst function, such as inorder traversal, preorder traversal and so on.
